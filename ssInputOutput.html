Shell script to demonstrate Input/Output

 

A fundamental aspect of working with Shell scripts is working with input, output, and returning those results. There are different editors like nano, vim, gedit etc where we can write our script and take input and produce output. The below program will take your name as input and output the same.

 

#!/bin/bash
echo "Enter your name: "
read name
echo
echo "Your name is: " $name
echo

 

If we store this program as test.sh, we will run it as: bash test.sh .
